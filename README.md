Simulador de Vida - Aventura Interactiva

Esta es una aplicación web de ficción interactiva donde el usuario toma decisiones que afectan el curso de su día.

Cómo configurarla:
1. Descarga los archivos del repositorio.
2. Abre el archivo `index.html` en cualquier navegador (Chrome, Firefox, Edge).
3. ¡Sigue las instrucciones en pantalla y elige tu destino!

Tecnologías usadas: HTML5, CSS3 y JavaScript vanilla.
